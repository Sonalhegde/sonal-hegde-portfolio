# Sonal Hegde — Interactive Portfolio

A cinematic, accessible portfolio for Sonal Hegde, focused on embedded systems,
IoT, cyber-physical systems, edge AI, and digital-twin research.

## Local development

```bash
npm install
npm run dev
npm run build
npm test
```

The hero defaults to a Canvas2D pixel/ASCII treatment and keeps an optional
lazy-loaded Spline scene behind the `ASCII / 3D` control. Pointer and hero motion
automatically reduce on touch devices and for `prefers-reduced-motion` visitors.
